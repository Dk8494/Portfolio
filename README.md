# Portfolio Website

This is the repository for my personal portfolio website, showcasing my projects and skills.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/Dk8494/Portfolio.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Building for Production

To build the project for production, run:
```bash
npm run build
```
